    Multi-modal MRI

    2204.13656
    发布时间 : 2023-09-13 16:58
    阅读 :

    MMML应用程序的三大类：1）交叉模态综合；2）多模态预测；3）跨模态推理
    MMML的五个基本概念：多模态信息表示、融合、对齐、翻译和协同学习。
    MMML还通过利用多模态数据的互补性、对齐性和冗余性提供了强大的功能。在这个框架内，互补性是指在描述一个实例时来自不同模态的信息组件之间的协同作用。对齐表示跨模态的信息组件之间的对应关系。冗余表示来自不同模态的信息分量具有相同的含义，增强了从它们学习的模型的鲁棒性。根据优化问题的定义，我们将涉及两个或三个数据模式的ML称为MMML，将涉及三个以上数据模式的ML称为“多模态学习”。

    多模态融合涉及来自多个模态的信息的组合，用于诸如分类或回归的预测任务。它通过三个关键方面提高预测质量：1）由于多模态信息冗余而提高的鲁棒性; 2）通过利用多模态信息互补性来提高准确性; 3）在没有特定模态的情况下进行预测的能力。鉴于近年来DNN在各种任务中的主导地位，我们的重点是DL内的多模态信息融合。基于先前的定义，我们将多模态信息融合分为三种类型：基于操作、双线性池化和基于图的融合，如图3所示。

    Bilinear Pooling Fusion(双线性池融合)
    双线性池化融合

    也称为基于张量的或基于双线性模型的融合，通过计算单峰特征向量的外或克罗内克乘积来集成单峰特征向量。这种方法捕获模态之间的高阶乘法交互，产生高度表达的多模态表示。特征向量通常被扩展为附加值1以保留单峰特征。尽管其优点，但处理由高维输入产生的大张量可能在计算上要求很高或不切实际。这促使采用低维近似，即，将高维张量分解为多个低维张量，以平衡表现力和计算效率

    多模态预测技术，如分类和回归，利用各种数据模态来提高理解的全面性和预测的准确性。领先模型通常使用联合表示对多模态数据进行编码，然后进行预测。模型之间的关键区别在于它们的编码策略和方法。

    MRI是磁共振成像(Megnetic Resonace Imaging)，原理是依据释放的能量（也可称信号）在物质内部不同结结构环境中有不同的衰减，通过外加的磁场检测所发射出的电磁波，可以得知构成该物体原子核的位置和种类，从而绘制物体内部的结构图像。

    在MRI成像的过程中，通过改变MR信号的影响因素，可以得到不同的影像，这些不同的影像就称之为序列。如根据T1值加权，可得到T1序列，根据T2值加权可得到T2序列。一个病例可以有多个序列，每个序列由许多切片组成。

    标签图中，高亮部分（白色）为增强肿瘤，包围着的一圈为坏疽部分，坏疽是指组织坏死后发腐呈现黑色，外面一圈（灰色）为浮肿区域。

    T1、T2是用于测量电磁波的物理量，他们可以作为成像的数据.根据T1来成像，就叫”T1加权成像”，临床工作中简称”T1”，T2同理。

    从下图可以看到，T1图像整体感官跟“临床图像”的“习惯配色风格”非常接近，白质是白的，灰质是灰的,脑脊液是黑的,所以T1图像可以看出各种断层解剖图,于是”T1看解剖结构”。

    T2信号跟水含量有关,许多病灶的T2信号要强于周围的正常组织,常呈高亮状态，因此从T2序列中可以清楚的看到病灶所处位置、大小。但对比标签图可发现，增强肿瘤周围的浮肿区域在T2序列中较为模糊，难以清晰勾勒出浮肿区域轮廓。

    基于翻译的方法使用生成对抗网络（GAN）模式来将图像从源模态翻译到目标模态。GAN由生成器和鉴别器组成，其中生成器学习生成可信数据，鉴别器惩罚生成器产生不切实际的结果。

    SAM 架构主要包含三个部分：图像编码器；提示编码器；以及掩码解码器。包含一个强大的图像编码器（计算图像嵌入），一个提示编码器（计算提示嵌入），一个轻量级掩码解码器（实时预测掩码）。在使用时，只需要对图像提取一次图像嵌入，可以在不同的提示下重复使用。给定一个图像嵌入，提示编码器和掩码解码器可以在浏览器中在~50毫秒内根据提示预测掩码。
    1.图像编码器：基于可扩展和强大的预训练方法，研究者使用MAE预训练的ViT，最小限度地适用于处理高分辨率输入。图像编码器对每张图像运行一次，在提示模型之前进行应用。

    2.提示编码器：考虑两组prompt：稀疏（点、框、文本）和密集（掩码）。研究者通过位置编码来表示点和框，并将对每个提示类型的学习嵌入和自由形式的文本与CLIP中的现成文本编码相加。密集的提示（即掩码）使用卷积进行嵌入，并通过图像嵌入进行元素求和。

    3.掩码解码器：掩码解码器有效地将图像嵌入、提示嵌入和输出token映射到掩码。

    笔记：

    提出了一种基于Transformer的多模态融合网络PT-Net。PT-Net的编码器从MRI的三种模态中提取并融合上下文信息，从粗到细，以获得跨模态和多尺度的肿瘤信息。
    解码器堆叠不同模态的特征图，并使用信道注意机制来校准多模态信息。其次，考虑到分割模型容易受到相似解剖结构的干扰而做出错误的预测，通过计算预测分割的激活区域与地面实况之间的距离，我们的损失函数迫使模型区分与肿瘤相似的解剖结构并做出正确的预测。

    网络的编码器构建在Transformer上，而解码器是基于CNN的架构。编码器提取和合并上下文信息，从三个modalityspecific腮腺MRI在不同的尺度。它可以更好地获得跨模态和多尺度的肿瘤信息。解码器堆叠各种模态的特征图，并使用信道注意机制来校准多模态信息。

    解剖结构感知的损失，用于指导深层模型区分腮腺解剖结构与肿瘤。（基于距离的损失函数）

    所提出的PT-Net的概述，其由基于变换器的编码器和基于CNN的解码器组成。卷积：卷积块。MFB：多模态融合块。SWB：移动窗口块。PM：补丁合并。ICM：信息校准模块。

    PT-Net遵循编码器-解码器架构的设计，使用CNN和Transformer来建模低级特征和长期依赖关系。

    在每个卷积层之后，我们执行GELU激活函数

    每个模态独立编码器的输入由主分支的一个模态和次分支的两个模态组成。

    在解码器组件中，我们主要使用CNN来形成我们的PT-Net的主干。这有两个原因。首先，CNN中的卷积运算比纯Transformer架构具有更强的提取局部信息的能力。其次，纯Transformer结构消耗大量计算资源，引入大量模型参数，使网络更难收敛。我们在下面描述模型的具体实现。

    /*
    归一化：
    Batch Normalization（批量归一化）
    批量归一化是用于训练深度学习模型的流行归一化方法之一。它通过在训练阶段稳定层输入的分布，使深度神经网络的训练更加快速和稳定。这种方法主要与内部协方差(internal covariate shift，ICS)有关，其中内部协方差指的是前几层更新时引起的层输入分布的变化。为了提高模型的训练效果，减少内部协方差偏移很重要。批量归一化在这里的作用是通过增加网络层，控制层输入的均值和方差来减少内部协方差偏移。

    优点：（1）批量归一化降低了内部协方差（ICS），加速了深度神经网络的训练。（2）减少了梯度对参数规模或其初始值的依赖性，从而提高了学习率，且没有发散（divergence）的风险。（3）批量归一化通过防止网络陷入饱和模式，使得使用饱和非线性（saturating nonlinearities）成为可能。

    Weight Normalization（权重归一化）
    权重归一化是对深度神经网络中的权重向量进行重新参数化的过程，其工作原理是将这些权重向量的长度与其方向解耦（decoupling）。简单来说，我们可以将权重归一化定义为提高神经网络模型的权重可优化性的方法。

    优点：（1）权重归一化可以改善优化问题的条件，并加快随机梯度下降的收敛速度。（2）它可以成功地应用于循环模型，如LSTMs，以及深度强化学习或生成模型

    Layer Normalization（层归一化）
    层归一化是一种提高各种神经网络模型训练速度的方法。与批归一化不同的是，这种方法直接从隐藏层内神经元的输入之和估计归一化统计量。层归一化基本上是为了克服批归一化的缺点，如依赖小批量等

    优点：通过在每个时间步长分别计算归一化统计量，可以很容易地将层归一化应用于递归神经网络。这种方法能有效稳定循环网络中的隐藏状态动态。

    Instance Normalization（实例归一化）
    实例归一化又称对比归一化（contrast normalization），与层归一化几乎相似。与批量归一化不同的是，实例归一化是应用于整批图像而不是单个图像。

    优点：这种归一化简化了模型的学习过程。实例归一化可以在测试时应用。

    */

    消融实验：
    消融实验（Ablation study）在深度学习领域是一种常见的实验方法，用于评估模型中各个组件对整体性能的影响。通过移除或替换模型中的某些组件，可以观察到模型性能的变化，从而理解这些组件的重要性和作用。

    消融实验有助于验证所提出的方法是否有效，以及各部分是否对模型的整体性能起到积极作用。虽然消融实验对于理解和评估深度学习模型很有帮助，但在发表论文时是否必须进行消融实验，取决于具体的研究问题、方法和期刊或会议的要求。以下是一些可能需要考虑消融实验的情况：

    1、如果你提出了一个新的算法或模型结构，并声称其中的某些组件对于性能提升至关重要，那么进行消融实验有助于证明这些组件的有效性和重要性。

    2、如果你对现有方法进行了改进或优化，并希望展示这些改进是如何影响模型性能的，消融实验可以帮助你阐明这一点。

    3、一些期刊或会议可能会要求在论文中包含消融实验，以确保所提出的方法具有可靠的实证支持。
